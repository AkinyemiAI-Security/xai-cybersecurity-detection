# Explainable AI for Cybersecurity 🔍

This project implements an Explainable Artificial Intelligence (XAI) system for detecting phishing attacks and providing transparent explanations for predictions.

## 🚀 Features
- Phishing detection using machine learning
- Explainable AI outputs (feature importance)
- Transparency in cybersecurity decision-making

## 🛠 Technologies Used
- Python
- Scikit-learn
- Pandas

## 🧠 Explainability Method
- Feature importance analysis using Random Forest

📄 Related Research
This project is inspired by my research on Explainable AI in Ethical Hacking and Cybersecurity.
